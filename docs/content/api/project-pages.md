## Endpoints

- `GET /public-api/projects/{project_id}/pages`
- `GET /public-api/projects/{project_id}/pages/{page_id}`
- `POST /public-api/projects/{project_id}/pages`

Use these endpoints to manage a project's "Your Pages" URLs.
